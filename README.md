## 🛠️ 使用方法

### 方式一：在 New API 后台嵌入

1. 登录青棠API后台
2. 进入 **设置** → **首页内容**
3. 粘贴以下链接：
   ```
   https://t479842598.github.io/newapi-homepage-tec/
   ```
4. 点击保存

### 方式二：独立部署

直接访问 GitHub Pages 地址即可。

### Logo 配置（可选）

在后台 **设置** → **Logo** 中配置：

**3D 立体风格：**
