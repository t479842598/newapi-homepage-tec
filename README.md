# 青棠API Homepage

现代深色主题首页模板，专为青棠API设计。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主页（支持 iframe 嵌入） |
| `logo-A1.svg` | Logo - 3D 立体风格 |
| `logo-C2.svg` | Logo - 多层线框风格 |

## 使用方法

### 首页设置

1. 在青棠API后台 → **设置** → **首页内容**
2. 粘贴 GitHub Pages 链接：
   ```
   https://tang74.top/
   ```
   或
   ```
   https://t479842598.github.io/newapi-homepage-tec/
   ```
3. 保存

### Logo 设置

在青棠API后台 → **设置** → **Logo**，粘贴以下链接：

**3D 立体风格：**
```
https://raw.githubusercontent.com/t479842598/newapi-homepage-tec/main/logo-A1.svg
```

**多层线框风格：**
```
https://raw.githubusercontent.com/t479842598/newapi-homepage-tec/main/logo-C2.svg
```
